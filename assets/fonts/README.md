# Fonts

Add these Playfair Display font files here for the onboarding screen:

- **PlayfairDisplay-Regular.ttf** – used as `Playfair`
- **PlayfairDisplay-Bold.ttf** – used as `PlayfairBold`

Download from [Google Fonts](https://fonts.google.com/specimen/Playfair+Display) (use "Download family" and copy the `.ttf` files into this folder).
